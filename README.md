# Five Nights at Frickbears 3

## What does randomization do to this game?

The ability to salvage each individual animatronic is locked behind receiving it in the multiworld, as well as upgrades from Upgrade Cadet, Giftboxes, and any other random items you might find while salvaging.

There is an option to randomise which animatronics are salvaged where, affecting your starting roster of animatronics as well. (Note that while salvaging, animatronics will still visually look like their unrandomised counterpart, and will send the location for their unrandomised counterpart, though it will give you the randomised animatronic for your night)

## Which items can be in another player's world?

You may receive: Animatronics, Upgrades, Giftboxes and Salvage Tools

## When the player receives an item, what happens?

When going into a night, you will only have access to the upgrades that you received before starting, if you want to use any upgrades received during the night, you will need to quit to the menu and reload your save.

Animatronics can be salvaged without receiving their corrosponding unlock item, but where's the fun in that? It's an honour system til I implement proper preventions

## Configuring your YAML file

### What is a YAML file and why do I need one?

Your YAML file contains a set of configuration options which provide the generator with information about how it should
generate your game. Each player of a multiworld will provide their own YAML file. This setup allows each player to enjoy
an experience customized for their taste, and different players in the same multiworld can all have different options.

### Where do I get a YAML file?

You can generate a YAML file using your Archipelago Launcher, search and open the "Generate Template Options" program 
then find "Five Night's at Frickbears 3.yaml" in the templates folder, open it in a text editor of your choice and change
all values to your choice.

### Connect to the MultiServer

1. Install the Frickbears 3.apworld

2. Install the Frickbears 3 Archipelago mod

3. Open Five Night's at Frickbears 3, create a new game then return to the main menu (this is to ensure the client does not read old saves)

4. Open the Frickbears 3 client in the Archipelago Launcher and connect to the Multiworld Server

5. Start a new game once you're ready
